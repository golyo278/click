Jquery/Javascript Click

Hide on Click

$(".hide").click(function() {
$(".hide").hide();
});

Alert on Click

$(".div").click(function() {
alert("BOO!");
});

Change Color on Click

$(".div").click(function() {
$( this ).css( "background-color", "red" );
});

Toggle on Click

$(".div").click(function(){
$('.img').toggle();
});

Play Audio on Click

var audio = new Audio('Bandrew.m4a');
$(".div").click(function() {
audio.play();
});

Animate on Click

$(".div").click(function(){
$(".div").animate({
left: '250px',
opacity: '0.5',
height: '150px',
width: '150px'});
});

Fade on Click

$(".div").click(function(){
$(".div").animate({
left: '250px',
opacity: '0.5',

});

Jquery is a javascript library that uses a simplified syntax. You must link a jquery library to your html file in order to use it. This can be done locally by linking a jquery file, or by linking to a URL of jqeury hosted on the web.
